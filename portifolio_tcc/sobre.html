Sobre o Projeto

Este projeto foi desenvolvido para auxiliar na gestão de tarefas, prazos e reuniões. Com uma interface intuitiva e moderna, ele permite que os usuários organizem suas atividades de forma eficiente e prática.

Objetivo

Facilitar a organização e o acompanhamento de tarefas importantes, garantindo maior produtividade e organização.

Tecnologias

HTML, CSS e JavaScript foram utilizados para criar uma interface moderna e responsiva.

Benefícios

Melhora na gestão do tempo, redução de atrasos e maior controle sobre as atividades diárias.
